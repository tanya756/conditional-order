常見問題

1. e智慧目前可交易什麼商品？

目前僅提供台股，不包含零股商品／指數類商品。

2. 使用e智慧下單和一般下單，有什麼不同？

使用e智慧下單可設定特定監控條件，當條件觸發時，將由系統執行委託下單，讓e智慧為您掌握交易機會。

3. 設定完成智慧單後， APP 是否要保持開啟才會執行？

不用！智慧單是在雲端洗價並監控，當設定條件符合後，立刻執行送單！

4. 若更換手機裝置，怎麼找回之前設定的智慧單？

只要在隨身e策略登入相同 ID 都可看到原先設定之智慧單內容。

5. 智慧單設定後是否可以修改條件？

設定後僅能刪除，無法修改。

6. 在同 ID 中使用多個帳號分別設定不同的智慧單，查詢時會分開顯示嗎？

會分開顯示，且可切換不同帳號查找該帳號設定的智慧單。

7. 同一個 ID 中，最多可設定幾組智慧單？

同一個 ID 中（不論有幾組帳號），最多可設定台股上限300組智慧單（刪除或已完成者不計入）。

8. 智慧單可查詢多久前的歷史監控紀錄？

可查詢近一年內的紀錄。
